# Analysing overlaps

## How to run this code : 

 - run command `conda env create -f env.yaml`
 - start the jupyter lab environment, command `jl`
 - go to jupyter notebook : OverlapsAnalysis.ipynb and select the environment : `takingroot`
 - run all cells
